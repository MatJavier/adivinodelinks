# Información de Enlace

Esta aplicación web permite obtener información de un enlace proporcionado por el usuario. Utiliza APIs para obtener datos de geolocalización y dirección IP, y muestra estos datos en el navegador.

## Características

- **Obtención de Información:** Proporciona datos como la dirección IP, ciudad, región, país y dominio del enlace proporcionado.
- **Diseño Responsivo:** La aplicación está diseñada para ser utilizada en dispositivos de diferentes tamaños, desde móviles hasta computadoras de escritorio.
- **Fácil de Usar:** Interfaz simple que permite al usuario ingresar el enlace y obtener la información con solo un clic.

## Tecnologías Utilizadas

- **HTML:** Para la estructura básica de la página web.
- **CSS (Tailwind CSS):** Para los estilos y el diseño responsivo.
- **JavaScript:** Para interactuar con las APIs y manipular el DOM.

## Cómo Usar

1. Ingresa el enlace que deseas analizar en el campo provisto.
2. Haz clic en el botón "Obtener Información".
3. La aplicación mostrará la información obtenida, incluyendo la dirección IP, ciudad, región, país y dominio del enlace.
